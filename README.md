Project Overview
This project presents an interactive Sales Analytics Dashboard built in Microsoft Excel using over 5,000+ sales records. The dashboard provides insights into revenue trends, product performance, and profitability to support data-driven decision-making.

 Objectives
Analyze historical sales data to identify trends and patterns
Track key business metrics like Total Sales, Profit, and Quantity
Identify top-performing and underperforming categories
Enable dynamic filtering for better business insights

Tools & Techniques Used
Microsoft Excel
Pivot Tables
Pivot Charts
XLOOKUP
Slicers & Filters
Data Cleaning & Transformation

 Key Features
 KPI Cards for:
Total Sales: ₹176,297.37
Total Profit: ₹2,263.03
Total Quantity: 1,906
 Monthly Sales Trend Analysis
Sales by Category Visualization
Category-wise Profit & Quantity Analysis
Interactive Slicers (Year, Sub-Category) for dynamic filtering

 Insights
Chairs generated the highest sales among all categories
Tables showed negative profit, indicating potential losses
Sales peaked during the later months of the year
Some sub-categories need pricing or cost optimization
